# Lab 2 - Shared-Memory Programming Model - K-means

Σκοπός της άσκησης ήταν η παραλληλοποίηση μιας υλοποίησης του K-means, για αρχιτεκτονικές κοινής μνήμης. Ακολουθούμε το μοντέλο προγραμματισμού κοινού χώρου διευθύνσεων, με χρήση του OpenMP. Πρέπει να εντοπίσουμε τα κομμάτια που παραλληλοποιούνται στον αλγόριθμο και να προσέξουμε για πιθανά race conditions.

Η πρώτη υλοποίηση είναι η Naive, δηλαδή η "πρώτη σκέψη" που θα έκανε κάποιος προσπαθώντας να παραλληλοποιήσει το πρόβλημα. Διατηρούμε έναν shared πίνακα του προβλήματος στον οποίο έχουν πρόσβαση όλα τα νήματα που εκτελούνται. Προχωρώντας λίγο την σκέψη μας, μπορούμε να σκεφτόυμε τρόπους να βελτιώσουμε την επίδοση του προβλήματος, με NUMA-aware τρόπους βελτιστοποίησης.
